EXERCÍCIO PRÁTICO

CRIE UM SISTEMA DE COBRANÇA ONDE O USUÁRIO ENTRE COM:
- DATA DE NASCIMENTO;
- O VALOR DA COMPRA;
- FORMA DE PAGAMENTO
	-DINHEIRO / DÉBITO
	-CRÉDITO / MASTERCARD
	-CRÉDITO / VISA
	-CRÉDITO / ELO

CALCULAR O VALOR TOTAL A SER PAGO, DE ACORDO COM A FORMA DE PAGAMENTO E IDADE DO COMPRADOR, INFORMANDO OS RESPECTIVOS VALORES:

=> SE TIVER MENOS DE 18 ANOS:
	-DINHEIRO / DÉBITO		 => 10% DE DESCONTO
=> SE TIVER 18 ANOS OU MAIS
	-CRÉDITO / MASTERCARD	 => 5% DE JUROS
	-CRÉDITO / VISA			 => 10% DE JUROS
	-CRÉDITO / ELO			 => 20% DE JUROS
